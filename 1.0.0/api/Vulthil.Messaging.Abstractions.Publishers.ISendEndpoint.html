Sends messages to a single, addressable destination (point-to-point delivery).

public interface ISendEndpoint
Extension Methods
ResultExtensions.ToResult<T>(T?, Error)

Remarks

Sends are addressed directly to the destination identified by Address and bypass the per-message-type exchange used by IPublisher. Topology for the destination queue is owned by the receiving service; the sender does not declare it.
