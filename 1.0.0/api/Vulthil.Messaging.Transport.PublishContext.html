Class PublishContext

Namespace
Vulthil.Messaging.Transport
Assembly
Vulthil.Messaging.dll

Mutable, write-through configuration captured for a single outgoing message. A transport creates one, passes it to the caller's configure callback as IPublishContext, then reads the resolved values to build its wire message. Metadata that maps to typed envelope fields is stored under reserved header keys and surfaced through the strongly-typed properties.
