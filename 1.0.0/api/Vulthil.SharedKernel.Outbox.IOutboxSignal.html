Interface IOutboxSignal

Namespace
Vulthil.SharedKernel.Outbox
Assembly
Vulthil.SharedKernel.Outbox.dll

Wakes the outbox background service when a transaction commits, so freshly-written outbox messages are relayed promptly instead of waiting for the next poll. Signals coalesce (at most one pending wake), and the periodic poll remains the correctness backstop for retries, other instances, and missed signals.
