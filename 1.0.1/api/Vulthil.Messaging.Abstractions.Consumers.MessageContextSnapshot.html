Class MessageContextSnapshot

Namespace
Vulthil.Messaging.Abstractions.Consumers
Assembly
Vulthil.Messaging.Abstractions.dll

An immutable, serializable snapshot of an IMessageContext's transport metadata captured at a point in time (e.g. when a Fault<TMessage> is produced). Unlike IMessageContext it carries no behavior and no live transport binding, so it round-trips cleanly through serialization.
