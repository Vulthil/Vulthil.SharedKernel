Class MessageEnvelope

Namespace
Vulthil.Messaging.Transport
Assembly
Vulthil.Messaging.dll

On-the-wire wrapper for messages carried by a Vulthil transport. The body of every produced message is a serialized MessageEnvelope; receivers fall back to bare JSON only for compatibility with non-Vulthil producers.

Remarks

All metadata that would otherwise live in transport-specific headers is promoted to first-class envelope fields, so the envelope is the source of truth and the wire format is transport-independent.
