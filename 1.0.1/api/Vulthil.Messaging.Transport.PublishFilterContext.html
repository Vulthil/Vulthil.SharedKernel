The context flowing through the publish/send filter pipeline for a single outgoing message. A filter may inspect the message and its resolved metadata, adjust Context, short-circuit the pipeline by not invoking next (e.g. to capture the message into a transactional outbox), or let it proceed to the transport terminal.
