An IOutboxStore that can purge old terminal (processed or dead-lettered) rows for the retention sweep. The EF Core outbox store implements this; the retention background service skips the sweep when the registered store does not.

DeleteProcessedAsync(DateTimeOffset, int, CancellationToken)

Deletes up to batchSize outbox rows that were processed or dead-lettered before olderThanUtc. Pending rows are never deleted.

Task<int> DeleteProcessedAsync(DateTimeOffset olderThanUtc, int batchSize, CancellationToken cancellationToken)
