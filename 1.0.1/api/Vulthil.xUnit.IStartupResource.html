Interface IStartupResource

Namespace
Vulthil.xUnit
Assembly
Vulthil.xUnit.dll

A test resource that performs one-time setup during host startup using the application's services — for example a Cosmos store creating its database and containers through the registered DbContext. Invoked once, after database migrations and before the application's background services, so the resource is ready before the first test runs.

public interface IStartupResource

ResultExtensions.ToResult<T>(T?, Error)

Remarks

Distinct from IResettableResource: startup setup runs once before any test, whereas a reset runs after each test. A resource may implement both — creating its store at startup and recreating it between tests.
