Interface IRestartableHostedService

Namespace
Vulthil.Extensions.Hosting
Assembly
Vulthil.Extensions.Hosting.dll

Marks an IHostedService whose execution can be stopped and started again cleanly, so infrastructure may pause it for the duration of an operation that must not run concurrently with it — for example a test harness resetting the database between tests, where a live database-polling relay would otherwise contend with the reset. Implement this only on services whose StartAsync(CancellationToken) / StopAsync(CancellationToken) are idempotent across repeated cycles (a BackgroundService whose ExecuteAsync re-runs cleanly).
