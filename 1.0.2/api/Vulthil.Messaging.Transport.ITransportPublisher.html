Interface ITransportPublisher

Namespace
Vulthil.Messaging.Transport
Assembly
Vulthil.Messaging.dll

A transport's raw publish terminal: it builds the wire envelope and sends it to the broker. The public IPublisher registered to callers is a filtering facade that runs the publish pipeline and then delegates to this terminal. Transports register their publisher under this interface instead of IPublisher.
