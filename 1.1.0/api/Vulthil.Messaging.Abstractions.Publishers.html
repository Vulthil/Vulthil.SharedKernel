Namespace Vulthil.Messaging.Abstractions.Publishers

Interfaces

IPublishContext

Provides a write-only context for configuring outgoing message properties. Every member sets a value on the message being published; nothing is read back through this interface.

IPublisher

Publishes messages to a message broker.

IRequestContext

Extends IPublishContext with request-specific configuration options, such as timeouts.

IRequester

Sends a request message and awaits a response from a remote consumer.

ISendEndpoint

Sends messages to a single, addressable destination (point-to-point delivery).

ISendEndpointProvider

Resolves ISendEndpoint instances for point-to-point message delivery.
