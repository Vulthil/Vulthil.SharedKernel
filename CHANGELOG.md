# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added

- Package validation (`EnablePackageValidation`) to catch breaking changes between releases.
- Source Link support for debugger source stepping.
- Symbol packages (`.snupkg`) published alongside NuGet packages.
- MIT license.
- Package metadata: descriptions, tags, project URL, and copyright.
- Public API analyzers for explicit API surface tracking.
- `CHANGELOG.md` following Keep a Changelog format.
- Expanded root `README.md` with package matrix, quick start, and documentation links.
