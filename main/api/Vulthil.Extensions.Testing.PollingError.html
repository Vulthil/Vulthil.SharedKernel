Remarks

When a WaitAsync<T>(TimeSpan, Func<Task<Result<T>>>, TimeSpan, CancellationToken) call times out, the returned PollingResult<T> provides the PollingError via PollingError:

var result = await Polling.WaitAsync(timeout, myFunc);
if (result.IsFailure && result.PollingError is { } pollingError)
{
    foreach (var error in pollingError.Errors)
    {
        // inspect each attempt's error
    }
}
