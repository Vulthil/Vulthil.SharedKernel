Remarks

Use the PollingError property to inspect individual errors from each polling attempt when the operation fails:

PollingResult result = await Polling.WaitAsync(timeout, myFunc);
if (result.IsFailure)
{
    foreach (var error in result.PollingError!.Errors)
    {
        // inspect each attempt's error
    }
}
