Remarks

Unlike IPublisher (which fans out a message via its configured exchange to any number of interested consumers), a send endpoint addresses a single, named destination — typically a specific queue on a specific service. The destination is identified by a Uri; the default scheme recognized by the RabbitMQ transport is queue:<name>.
