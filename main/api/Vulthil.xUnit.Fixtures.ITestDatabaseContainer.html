MigrateDatabase(IServiceProvider)

Ensures pending EF Core migrations are applied to the containerized database. Invoked during host startup, before the application's own background services run, so the schema exists in time; only migrations that remain pending are applied, and a concurrent migrator (such as an application that migrates itself) is tolerated rather than fought.
